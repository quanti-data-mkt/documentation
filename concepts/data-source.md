# Data Source

A **Data Source** is a connector type available in QUANTI: — for example, Meta Ads, Google Ads, GA4, or TikTok Ads. Each Data Source represents a specific platform or service you can connect to.

All QUANTI: plans give you access to the **full catalog of Data Sources** — there is no restriction on which connector types you can use, regardless of your plan.

From a single Data Source, you can create multiple [Connected Accounts](connected-account.md) — for instance, connecting several Meta Ads ad accounts using the same Meta Ads Data Source.
